# Projet COBOL – Gestion d’un livret d’épargne

## Description

Ce projet simule la gestion d’un livret d’épargne avec :
- Un capital initial  
- Un taux d’intérêt annuel  
- Une durée d’épargne  

Le programme calcule :
- Les intérêts générés  
- L’évolution du capital  
- Le montant final  

## Objectifs pédagogiques

- Implémenter des calculs d’intérêts  
- Manipuler des boucles  
- Gérer les données financières  
- Structurer un programme COBOL  

## Fonctionnalités

- Saisie du capital de départ  
- Saisie du taux d’intérêt  
- Saisie de la durée  
- Calcul du capital final  
- Affichage des résultats  

## Technologies utilisées

- COBOL  
- Logique métier financière  

## Améliorations prévues

- Gestion de plusieurs comptes  
- Sauvegarde dans un fichier  
- Traitements batch  
- Connexion DB2  

## Contexte

Projet développé dans le cadre de mon auto-formation COBOL pour renforcer mes compétences en logique financière et programmation mainframe.
