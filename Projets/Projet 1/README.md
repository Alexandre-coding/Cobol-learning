# Projet COBOL – Calcul de prêt immobilier

## Description

Ce projet permet de simuler le calcul des mensualités d’un prêt immobilier à partir de :
- Le montant du prêt  
- Le taux d’intérêt  
- La durée du prêt  

Le programme calcule :
- Les mensualités  
- Le coût total du crédit  
- Les intérêts payés  

## Objectifs pédagogiques

- Manipuler les calculs financiers en COBOL  
- Utiliser les variables numériques  
- Structurer un programme procédural  
- Gérer les entrées utilisateur  

## Fonctionnalités

- Saisie du montant du prêt  
- Saisie du taux d’intérêt  
- Saisie de la durée  
- Calcul automatique des mensualités  
- Affichage des résultats  

## Technologies utilisées

- COBOL  
- Environnement Mainframe / COBOL standard  

## Améliorations prévues

- Lecture des données depuis un fichier  
- Génération d’un tableau d’amortissement  
- Traitement batch  
- Intégration DB2 (optionnel)  

## Contexte

Projet réalisé dans le cadre de mon auto-formation COBOL pour une reconversion professionnelle vers le développement mainframe.
