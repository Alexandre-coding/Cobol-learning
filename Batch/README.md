# Batch

Ce dossier contient des exemples de programmes orientés traitements batch.

Objectifs :
- Comprendre la logique des traitements batch
- Simuler des programmes exécutés en production
- Appréhender la notion de traitement automatisé

Contenu :
- Programmes COBOL exécutés sans interaction utilisateur
- Exemples pédagogiques de traitements batch
