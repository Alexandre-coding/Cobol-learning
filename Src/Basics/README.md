# Basics

Ce dossier contient des programmes COBOL de base réalisés dans le cadre
de l’apprentissage du langage.

Objectifs :
- Comprendre la structure d’un programme COBOL
- Manipuler les variables
- Utiliser les instructions de base (DISPLAY, ACCEPT, IF, PERFORM)

Contenu :
- Programmes simples (Hello World, variables, conditions, boucles)
