# Projets COBOL

Ce dossier contient mes projets principaux réalisés en COBOL dans le cadre de mon auto-formation et de ma reconversion vers le développement mainframe.

L’objectif de ces projets est de mettre en pratique :
- La syntaxe COBOL
- Les calculs financiers
- La gestion de fichiers
- Les traitements batch
- La logique métier

Chaque projet dispose de son propre dossier avec :
- Le code source COBOL
- Un README expliquant le fonctionnement
- Les fonctionnalités principales

## Liste des projets

### 1. Calcul de prêt immobilier
Simulation du calcul des mensualités d’un prêt avec intérêts.

Compétences utilisées :
- Calculs financiers  
- COBOL procédural  
- Gestion des entrées utilisateur  

### 2. Gestion d’un livret d’épargne
Simulation de la gestion d’un livret avec calcul des intérêts.

Compétences utilisées :
- Calcul d’intérêts  
- Logique métier  
- Traitements COBOL  

## Objectif professionnel

Ces projets s’inscrivent dans mon projet de reconversion vers un poste de Développeur COBOL / Mainframe, avec une orientation vers les environnements bancaires et assurantiels.
