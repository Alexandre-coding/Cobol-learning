# COBOL Learning Projects

Ce dépôt regroupe mes travaux pratiques réalisés dans le cadre de ma reconversion
vers le développement COBOL, après une première expérience en environnement AS/400 (RPG).

L’objectif est de consolider les bases du langage COBOL et de mieux comprendre
la logique des traitements batch et des applications legacy.

## Contenu

- `Basics/`  
  Programmes COBOL de base : affichage, variables, structures simples.

- `Algorithms/`  
  Exercices de logique et d’algorithmique (exemple : calcul de nombres premiers).

## Technologies et outils

- COBOL
- GnuCOBOL
- VS Code
- Git / GitHub

## Statut

Projet pédagogique en cours d’évolution.
